# Google Sheet Column Names Guide

To display proper English names on the website, use these exact column names in your Google Sheet:

## Required Column Names:

1. **Year** - The year/period column (e.g., 2005, 2006, 2007...)
   - Alternative names that also work: "Period", "Date", "vuosi"

2. **Firms** - Number of firms/companies
   - Alternative names: "Companies", "Startups", "Number of Firms"

3. **Revenue** - Revenue/turnover amount
   - Alternative names: "Turnover", "Sales", "Liikevaihto"

4. **Employees** - Total number of employees
   - Alternative names: "Employment", "Jobs", "Workers", "Työlliset"

5. **Employees in Finland** - Number of employees in Finland
   - Alternative names: "Employees Finland", "Työlliset_suomessa", "Työlliset suomessa"

## Example Google Sheet Structure:

| Year | Firms | Revenue | Employees | Employees in Finland |
|------|-------|---------|-----------|---------------------|
| 2005 | 87    | 523511111.7 | 3367 | 1399 |
| 2006 | 97    | 578353464.6 | 3604 | 1848 |
| 2007 | 110   | 890006661.4 | 3852 | 2237 |

## Notes:

- Column names are case-insensitive (Year = year = YEAR)
- The first row should contain the column headers
- Make sure there are no extra spaces in column names
- The system will automatically detect these columns and display them with proper English names on the website

## Current Finnish Column Names (if you want to keep them):

If you prefer to keep Finnish names, the system will still work, but the website will display:
- "Vuosi" instead of "Year"
- "Liikevaihto" instead of "Revenue"
- "Työlliset" instead of "Employees"
- "Yritykset" instead of "Firms"
- "Työlliset_suomessa" instead of "Employees in Finland"
